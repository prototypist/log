# log

comically simple wrapper for console logging, using the lovely [chalk](https://www.npmjs.com/package/chalk) package for coloring.

### installation

`yarn add https://github.com/prototypist/log.git`

### usage

```
const log = require('log');

log.system('system');
log.info('info');
log.success('sucess');
log.warning('warning');
log.error('error');
```
